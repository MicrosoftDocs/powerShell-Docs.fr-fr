# <a name="wraptables-element-format"></a>WrapTables, élément (Format)

Spécifie que les données dans une cellule de tableau sont déplacées vers la ligne suivante si les données sont supérieure à la largeur de la colonne.

Configuration élément (Format) DefaultSettings (Format) WrapTables élément (Format)

## <a name="syntax"></a>Syntaxe

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Éléments et attributs

Les sections suivantes décrivent les attributs et éléments enfants de l’élément parent le `WrapTables` élément.

### <a name="attributes"></a>Attributes

Aucune.

### <a name="child-elements"></a>Éléments enfants

Aucune.
