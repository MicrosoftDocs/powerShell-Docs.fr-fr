# <a name="text-element-for-customitem-for-controls-for-configuration-format"></a>Text, élément pour CustomItem pour Controls pour Configuration (Format)

Spécifie le texte qui est ajouté aux données qui est affiché par le contrôle, par exemple une étiquette, des crochets pour délimiter les données et les espaces à mettre en retrait les données. Cet élément est utilisé lors de la définition d’un contrôle commun qui peut être utilisé par toutes les vues dans le fichier de mise en forme.
