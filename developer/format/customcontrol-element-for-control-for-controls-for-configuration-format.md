# <a name="customcontrol-element-for-control-for-controls-for-configuration-format"></a>CustomControl, élément pour Control pour Controls (Format)

Définit un contrôle. Cet élément est utilisé lors de la définition d’un contrôle commun qui peut être utilisé par toutes les vues dans le fichier de mise en forme.

Élément de contrôles (Format) d’élément de configuration de l’élément de contrôle de Configuration (Format) pour les contrôles d’élément de CustomControl Configuration (Format) pour le contrôle de Configuration (Format)

## <a name="syntax"></a>Syntaxe

```xml
<CustomControl>
  <CustomEntries>...</CustomEntries>
</CustomControl>
```

## <a name="attributes-and-elements"></a>Éléments et attributs

Les sections suivantes décrivent les attributs et éléments enfants de l’élément parent le `CustomControl` élément. Cet élément doit avoir au moins un élément enfant. Il n’existe aucune limite maximale pour le nombre d’éléments enfants qui peuvent être spécifiées.
