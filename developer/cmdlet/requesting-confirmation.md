# <a name="requesting-confirmation"></a>Demandes de confirmation

Cette section traite des messages de confirmation qui peuvent être affichées avant une applet de commande, fonction, ou le fournisseur effectue une action.

## <a name="in-this-section"></a>Dans cette section

[Demande de processus de Confirmation pour les commandes](./requesting-confirmation-from-cmdlets.md) présente le processus que les applets de commande, les fonctions et les fournisseurs doivent suivre pour demander une confirmation avant de leur apportent une modification au système.

[Confirmation de demande des utilisateurs](./users-requesting-confirmation.md) explique comment les utilisateurs peuvent apporter une confirmation de demande d’applet de commande, fonction ou fournisseur lors de la [System.Management.Automation.Cmdlet.ShouldProcess](/dotnet/api/System.Management.Automation.Cmdlet.ShouldProcess) méthode est appelée.

[Messages de confirmation](./confirmation-messages.md) fournit des exemples de messages de confirmation différents qui peuvent être affichés.
