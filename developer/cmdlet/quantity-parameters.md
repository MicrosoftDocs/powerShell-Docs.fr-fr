# <a name="quantity-parameters"></a>Paramètres de quantité

Le tableau suivant répertorie les noms recommandés pour les paramètres de la quantité.

|Paramètre|Fonctionnalités|
|---|---|
|**Tous les**<br>Type de données : Booléen|Mettre en œuvre de ce paramètre afin que `true` indique que toutes les ressources doivent être exploitées au lieu d’un sous-ensemble de la valeur par défaut des ressources. Mettre en œuvre de ce paramètre afin que `false` indique un sous-ensemble des ressources.|
|**Allocation**<br>Type de données : Int32|Implémentez ce paramètre afin que l’utilisateur peut spécifier le nombre d’éléments à allouer.|
|**BlockCount**<br>Type de données : Int64|Implémentez ce paramètre afin que l’utilisateur peut spécifier le nombre de blocs.|
|**Count**<br>Type de données : Int64|Implémentez ce paramètre afin que l’utilisateur peut spécifier le nombre.|
|**Portée**<br>Type de données : Mot clé|Implémentez ce paramètre afin que l’utilisateur peut spécifier l’étendue à utiliser.|
