# <a name="standard-cmdlet-parameter-names-and-types"></a>Noms et types des paramètres des applets de commande standard

Noms de paramètres d’applet de commande doivent être cohérents entre les applets de commande que vous concevez. Les rubriques suivantes répertorient les noms de paramètres, nous vous recommandons de qu'utiliser lorsque vous déclarez des paramètres de l’applet de commande. Les rubriques décrivent également le type de données recommandés et les fonctionnalités de chaque paramètre.
