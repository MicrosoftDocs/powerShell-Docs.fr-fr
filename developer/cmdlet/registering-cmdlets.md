Les rubriques de cette section décrivent les modules et composants logiciel enfichables et comment utiliser les modules et composants logiciel enfichables de proposer des applets de commande dans une session Windows PowerShell.

## <a name="in-this-section"></a>Dans cette section

[Modules et composants logiciel enfichables](./modules-and-snap-ins.md) décrit les différences entre l’inscription des applets de commande via des modules et composants logiciel enfichables.

[Comment les applets de commande s’inscrire à l’aide de Modules](./how-to-import-cmdlets-using-modules.md) explique comment inscrire les applets de commande à l’aide de modules.

[Comment créer un Windows PowerShell Snap-in](./how-to-create-a-windows-powershell-snap-in.md) explique comment inscrire les applets de commande à l’aide des composants logiciel enfichables.
