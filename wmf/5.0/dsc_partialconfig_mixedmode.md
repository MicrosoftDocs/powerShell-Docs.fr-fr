# <a name="support-for-mixed-refreshmode"></a>Prise en charge du mode d’actualisation mixte

Quand vous utilisez des configurations partielles, vous pouvez maintenant définir chaque configuration partielle avec un `RefreshMode`.
Pour plus d’informations, consultez [Configurations partielles](https://msdn.microsoft.com/powershell/dsc/partialconfigs).
