## <a name="description"></a><span data-ttu-id="ad262-105">Description</span><span class="sxs-lookup"><span data-stu-id="ad262-105">Description</span></span>

<span data-ttu-id="ad262-106">L’applet de commande Update-Module installe une version plus récente d’un module Windows PowerShell qui a été installé à partir de la galerie en ligne en exécutant Install-Module sur l’ordinateur local.</span><span class="sxs-lookup"><span data-stu-id="ad262-106">The Update-Module cmdlet installs a newer version of a Windows PowerShell module that was installed from the online gallery by running Install-Module on the local computer.</span></span>
