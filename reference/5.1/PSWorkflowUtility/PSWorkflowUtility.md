# Module PSWorkflowUtility

## Description

Cette section contient les rubriques d'aide pour les commandes exportées par le module PSWorkflowUtility Windows PowerShell. Ce module contient un workflow qui prend en charge la fonctionnalité de workflow Windows PowerShell.

## Applets de commande PSWorkflowUtility

### [Invoke-AsWorkflow](Invoke-AsWorkflow.md)
Exécute une commande ou une expression comme un workflow Windows PowerShell.
