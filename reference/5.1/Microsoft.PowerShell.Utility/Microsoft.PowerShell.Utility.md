# Module Microsoft.PowerShell.Utility

## Description

Cette section contient les rubriques d’aide pour les applets de commande qui sont installées avec le module Microsoft. PowerShell. Utility. Ce module contient des applets de commande qui gèrent les fonctionnalités de base de PowerShell.
