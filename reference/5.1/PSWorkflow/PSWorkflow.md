# Module PSWorkflow

## Description

Cette section contient les rubriques d'aide relatives aux applets de commande qui sont installées avec le module PSWorkflow Windows PowerShell, qui contient des applets de commande qui prennent en charge la fonctionnalité de workflow Windows PowerShell.

## Applets de commande PSWorkflow

### [New-PSWorkflowExecutionOption](New-PSWorkflowExecutionOption.md)
Crée un objet qui contient les options de configuration de session pour les sessions de workflow.

### [New-PSWorkflowSession](New-PSWorkflowSession.md)
Crée une session de workflow.
