# Module ISE

## Description

Cette section contient les rubriques d’aide pour les applets de commande installées avec le module Environnement d’écriture de scripts intégré de Windows PowerShell (ISE). Le module Windows PowerShell ISE contient des applets de commande qui ajoutent des fonctionnalités à Windows PowerShell ISE.

## Applets de commande ISE

### [Get-IseSnippet](Get-IseSnippet.md)
Obtient les extraits de code créés par l'utilisateur.

### [Import-IseSnippet](Import-IseSnippet.md)
Importe les extraits de code de l'environnement ISE dans la session active.

### [New-IseSnippet](New-IseSnippet.md)
Crée un extrait de code Windows PowerShell ISE.
