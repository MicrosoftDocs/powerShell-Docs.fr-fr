# Module Microsoft.PowerShell.Archive

## Description

Cette section contient les rubriques d’aide pour les applets de commande qui sont installées avec le module PowerShell Microsoft. PowerShell. archive. Le module Archive contient des applets de commande qui vous permettent de créer et d’extraire des fichiers d’archive ou ZIP.

## Applets de commande Microsoft. PowerShell. archive

### [Compress-Archive](Compress-Archive.md)
Crée une archive compressée, ou un fichier compressé, à partir des fichiers et répertoires spécifiés.

### [Expand-Archive](Expand-Archive.md)
Extrait des fichiers à partir d’un fichier d’archive (zippé) spécifié.
