# Module ThreadJob

## Description
Ce module étend le BackgroundJob PowerShell existant pour inclure une nouvelle tâche **ThreadJob** basée sur un thread. Il s’agit d’une solution plus légère pour exécuter des scripts PowerShell simultanés qui fonctionnent dans l’infrastructure de travaux PowerShell existante.

## Applets de commande ThreadJob

### [Start-ThreadJob](Start-ThreadJob.md)
Crée des travaux en arrière-plan similaires à l’applet de commande `Start-Job` .
