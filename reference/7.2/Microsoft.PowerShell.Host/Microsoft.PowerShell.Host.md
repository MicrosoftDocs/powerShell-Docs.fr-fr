# Module Microsoft.PowerShell.Host

## Description

Cette section contient les rubriques d’aide pour les applets de commande qui sont installées avec le module PowerShell Microsoft. PowerShell. Host. Le module Host contient des applets de commande qui gèrent les données issues des programmes hôtes.

## Applets de commande Microsoft.PowerShell.Host

### [Start-Transcript](Start-Transcript.md)
Crée un enregistrement de tout ou partie d’une session PowerShell dans un fichier texte.

### [Stop-Transcript](Stop-Transcript.md)
Arrête une transcription.
