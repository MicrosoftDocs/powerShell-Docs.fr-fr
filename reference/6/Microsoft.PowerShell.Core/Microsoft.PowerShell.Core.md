# Module Microsoft.PowerShell.Core

## Description

Cette section contient les rubriques d’aide pour les applets de commande installées avec le module Microsoft. PowerShell. Core PowerShell. Le module principal contient des applets de commande et des fournisseurs qui gèrent les fonctionnalités de base de PowerShell.

## Applets de commande Microsoft. PowerShell. Core

### [Add-History](Add-History.md)
Ajoute des entrées à l'historique de session.

### [Clear-History](Clear-History.md)
Supprime des entrées de l’historique de commande de session PowerShell.

### [Clear-Host](Clear-Host.md)
Efface l'affichage dans le programme hôte.

### [Connect-PSSession](Connect-PSSession.md)
Se reconnecte aux sessions déconnectées.

### [Debug-Job](Debug-Job.md)
Débogue un travail d’arrière-plan, distant ou PowerShell en cours d’exécution.

### [Disable-ExperimentalFeature](Disable-ExperimentalFeature.md)
Désactivez une fonctionnalité expérimentale au démarrage de la nouvelle instance de PowerShell.

### [Disable-PSRemoting](Disable-PSRemoting.md)
Empêche les points de terminaison PowerShell de recevoir des connexions à distance.

### [Disable-PSSessionConfiguration](Disable-PSSessionConfiguration.md)
Désactive les configurations de session sur l'ordinateur local.

### [Disconnect-PSSession](Disconnect-PSSession.md)
Se déconnecte d'une session.

### [Enable-ExperimentalFeature](Enable-ExperimentalFeature.md)
Activez une fonctionnalité expérimentale au démarrage de la nouvelle instance de PowerShell.

### [Enable-PSRemoting](Enable-PSRemoting.md)
Configure l'ordinateur pour recevoir des commandes à distance.

### [Enable-PSSessionConfiguration](Enable-PSSessionConfiguration.md)
Active les configurations de session sur l'ordinateur local.

### [Enter-PSHostProcess](Enter-PSHostProcess.md)
Se connecte à et entre dans une session interactive avec un processus local.

### [Enter-PSSession](Enter-PSSession.md)
Démarre une session interactive avec un ordinateur distant.

### [Exit-PSHostProcess](Exit-PSHostProcess.md)
Ferme une session interactive avec un processus local.

### [Exit-PSSession](Exit-PSSession.md)
Termine une session interactive avec un ordinateur distant.

### [Export-ModuleMember](Export-ModuleMember.md)

### [Get-Command](Get-Command.md)

### [Get-ExperimentalFeature](Get-ExperimentalFeature.md)

### [Get-History](Get-History.md)

### [Get-Job](Get-Job.md)

### [Get-Module](Get-Module.md)

### [Get-PSHostProcessInfo](Get-PSHostProcessInfo.md)

### [Get-PSSession](Get-PSSession.md)

### [Get-PSSessionCapability](Get-PSSessionCapability.md)

### [Get-PSSessionConfiguration](Get-PSSessionConfiguration.md)

### [Module d’importation](Import-Module.md)

### [Invoke-Command](Invoke-Command.md)

### [Invoke-History](Invoke-History.md)

### [New-Module](New-Module.md)
