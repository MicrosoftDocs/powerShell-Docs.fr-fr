# PSConsoleHostReadLine

## SYNOPSIS
Cette fonction est le point d’entrée principal pour PSReadLine.

## SYNTAX

```
PSConsoleHostReadLine
```

## Description

`PSConsoleHostReadLine` est le point d’entrée principal pour le module PSReadLine. L’hôte de la console PowerShell charge automatiquement le module PSReadLine et appelle cette fonction. Dans des conditions de fonctionnement normales, cette fonction n’est pas destinée à être utilisée à partir de la ligne de commande.

Le point `PSConsoleHostReadLine` d’extension est spécial pour l’hôte de la console. L’hôte appelle un alias, une fonction ou un script portant ce nom. PSReadLine définit cette fonction de manière à ce qu’elle soit appelée à partir de l’hôte de la console.
