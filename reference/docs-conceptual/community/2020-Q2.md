# <a name="whats-new-in-powershell-docs-for-2020-q2"></a>Nouveautés de PowerShell-Docs pour le deuxième trimestre 2020

Cet article présente les principales modifications apportées à Docs au cours du mois précédent et met à l’honneur les contributions de la communauté.

Aidez-nous à améliorer la documentation afin de mieux répondre à vos besoins. Pour savoir comment commencer, lisez le [Guide du contributeur][contrib].

## <a name="2020-june-highlights"></a>Points forts de juin 2020

- Nouvelle documentation PowerShell
  - Publication d’un nouveau contenu [Notions de base de PowerShell](../learn/ps101/00-introduction.md) auquel a contribué Mike F. Robbins
  - Ajout de deux billets de blog récents de Rob Holt à la documentation **Création de scripts et développement**
    - [Choix du bon package NuGet PowerShell pour votre projet .NET](/powershell/scripting/dev-cross-plat/choosing-the-right-nuget-package)
    - [Résolution des conflits de dépendances d’assembly de module PowerShell](/powershell/scripting/dev-cross-plat/resolving-dependency-conflicts)
- Maintenance de la documentation
  - Archivage du contenu ancien dans [https://aka.ms/PSLegacyDocs](https://aka.ms/PSLegacyDocs)
    - Contenu du SDK PowerShell Web Access
    - Contenu du SDK des workflows PowerShell
  - Mises à jour pour le contenu 7.1-preview

- Statistiques GitHub
  - Fusion de 83 demandes de tirage (15 provenant de la communauté)
  - 68 problèmes ouverts (52 provenant de la communauté)
  - 74 problèmes fermés

## <a name="2020-may-highlights"></a>Points forts de mai 2020

- Nouvelle documentation PowerShell

    - [Tout ce que vous voulez savoir sur la substitution de chaînes](../learn/deep-dives/everything-about-string-substitutions.md)
    - [Tout ce que vous voulez savoir sur if/then/else](../learn/deep-dives/everything-about-if.md)
    - [Tout ce que vous voulez savoir sur switch](../learn/deep-dives/everything-about-switch.md)
    - [Tout ce que vous voulez savoir sur les exceptions](../learn/deep-dives/everything-about-exceptions.md)

    - Nouveau contenu sous **Apprentissage de PowerShell**
