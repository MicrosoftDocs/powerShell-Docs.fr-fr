# <a name="whats-new-in-powershell-docs-for-2020-q3"></a>Nouveautés de la documentation PowerShell du troisième trimestre 2020

Cet article présente les principales modifications apportées à Docs au cours du mois précédent et met à l’honneur les contributions de la communauté.

Aidez-nous à améliorer la documentation afin de mieux répondre à vos besoins. Pour savoir comment commencer, lisez le [Guide du contributeur][contrib].


## <a name="2020-september-highlights"></a>Principaux points de septembre 2020

- Maintenance de la documentation
  - Mises à jour pour le contenu 7.1-preview

- Présentation de la communauté
  - Comment contribuer à la documentation pour RTPUG - https://www.youtube.com/watch?v=0_DEB61YOMc

- Statistiques GitHub
  - Fusion de 41 demandes de tirage (9 de la communauté)
  - 52 problèmes ouverts (47 de la communauté)
  - 51 problèmes fermés

## <a name="2020-july-highlights"></a>Points forts de juillet 2020

- Nouvelle documentation PowerShell
  - Réactivation de l’[ancienne documentation ETS](/powershell/scripting/developer/ets/overview) - Ajout de 7 articles
  - Ajout d’un article sur la [création d’une aide modifiable à l’aide de PlatyPS](/powershell/scripting/dev-cross-plat/create-help-using-platyps)
- Maintenance de la documentation
  - Mises à jour pour le contenu 7.1-preview
  - Mise à jour de l’en-tête de page - Simplification du choix de menu et ajout du bouton de téléchargement
  - Correction de plusieurs problèmes liés à `Update-Help`
    - L’aide pour les modules PSDesiredStateConfiguration et ThreadJob peut désormais être téléchargée
    - Publication de l’aide modifiable pour PowerShell 7.1 preview
    - L’aide modifiable pour PowerShell 5.1 comprend maintenant des rubriques À propos

- Statistiques GitHub
  - Fusion de 99 demandes de tirage (29 provenant de la communauté)
  - 51 problèmes ouverts (44 provenant de la communauté)
  - 71 problèmes fermés
