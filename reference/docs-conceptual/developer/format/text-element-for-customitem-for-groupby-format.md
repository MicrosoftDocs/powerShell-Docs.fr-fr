# <a name="text-element-for-customitem-for-groupby-format"></a>Text, élément pour CustomItem pour GroupBy (Format)

Spécifie le texte qui est ajouté aux données affichées par le contrôle, telles qu’une étiquette, les crochets pour encadrer les données et les espaces pour mettre en retrait les données. Cet élément est utilisé lors de la définition du mode d’affichage d’un nouveau groupe d’objets.
