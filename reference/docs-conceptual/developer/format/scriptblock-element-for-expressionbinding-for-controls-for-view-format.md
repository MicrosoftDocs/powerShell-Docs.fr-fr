# <a name="scriptblock-element-for-expressionbinding-for-controls-for-view-format"></a>ScriptBlock, élément pour ExpressionBinding pour Controls pour View (Format)

Spécifie le script dont la valeur est affichée par le contrôle. Cet élément est utilisé lors de la définition de contrôles qui peuvent être utilisés par une vue.
