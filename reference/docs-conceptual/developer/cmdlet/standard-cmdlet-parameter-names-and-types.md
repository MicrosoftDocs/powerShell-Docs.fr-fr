# <a name="standard-cmdlet-parameter-names-and-types"></a>Noms et types des paramètres des applets de commande standard

Les noms de paramètres d’applet de commande doivent être cohérents entre les applets de commande que vous concevez. Les rubriques suivantes répertorient les noms de paramètres que nous vous recommandons d’utiliser lorsque vous déclarez des paramètres d’applet de commande. Les rubriques décrivent également le type de données et les fonctionnalités recommandées pour chaque paramètre.
