# <a name="requesting-confirmation"></a>Demandes de confirmation

Cette section décrit les messages de confirmation qui peuvent être affichés avant qu’une applet de commande, une fonction ou un fournisseur n’effectue une action.

## <a name="in-this-section"></a>Dans cette section

[Demande du processus de confirmation pour les commandes](./requesting-confirmation-from-cmdlets.md) Décrit le processus que les applets de commande, les fonctions et les fournisseurs doivent suivre pour demander une confirmation avant d’apporter une modification au système.

[Utilisateurs demandant confirmation](./users-requesting-confirmation.md) Explique comment les utilisateurs peuvent créer une applet de commande, une fonction ou une confirmation de demande de fournisseur lors de l’appel de la méthode [System. Management. Automation. applet de commande. ShouldProcess](/dotnet/api/System.Management.Automation.Cmdlet.ShouldProcess) .

[Messages de confirmation](./confirmation-messages.md) Fournit des exemples des différents messages de confirmation qui peuvent être affichés.
