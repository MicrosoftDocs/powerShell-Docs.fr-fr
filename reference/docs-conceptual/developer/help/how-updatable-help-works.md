# <a name="how-updatable-help-works"></a><span data-ttu-id="22b4c-102">Fonctionnement de l’aide actualisable</span><span class="sxs-lookup"><span data-stu-id="22b4c-102">How Updatable Help Works</span></span>

<span data-ttu-id="22b4c-103">Cette rubrique explique comment mettre à jour l’aide pour traiter le fichier XML et les fichiers CAB HelpInfo pour chaque module, et comment installer l’aide mise à jour pour les utilisateurs.</span><span class="sxs-lookup"><span data-stu-id="22b4c-103">This topic explains how Updatable Help processes the HelpInfo XML file and CAB files for each module, and installs updated help for users.</span></span>

1. <span data-ttu-id="22b4c-106">`Update-Help`Obtient le fichier XML HelpInfo distant à partir de l’emplacement spécifié par la valeur de la clé **HelpInfoURI** dans le manifeste de module et valide le fichier par rapport au schéma.</span><span class="sxs-lookup"><span data-stu-id="22b4c-106">`Update-Help` gets the remote HelpInfo XML file from the location specified by the value of the **HelpInfoURI** key in the module manifest and validates the file against the schema.</span></span> <span data-ttu-id="22b4c-107">(Pour afficher le schéma, consultez [schéma XML HelpInfo](./helpinfo-xml-schema.md).) `Update-Help`Recherche ensuite un fichier XML HelpInfo local pour le module dans le répertoire du module sur l’ordinateur de l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="22b4c-107">(To view the schema, see [HelpInfo XML Schema](./helpinfo-xml-schema.md).) 1. <span data-ttu-id="22b4c-108">`Update-Help`compare le numéro de version des fichiers d’aide pour la culture d’interface utilisateur spécifiée dans les fichiers HelpInfo XML distants et locaux pour le module.</span><span class="sxs-lookup"><span data-stu-id="22b4c-108">`Update-Help` compares the version number of the help files for the specified UI culture in the remote and local HelpInfo XML files for the module.</span></span> <span data-ttu-id="22b4c-109">Si le numéro de version du fichier distant est supérieur au numéro de version du fichier local, ou s’il n’existe aucun fichier XML HelpInfo local pour le module, `Update-Help` prépare le téléchargement des nouveaux fichiers d’aide.</span><span class="sxs-lookup"><span data-stu-id="22b4c-109">If the version number on the remote file is greater than version number on the local file, or if the there is no local HelpInfo XML file for the module, `Update-Help` prepares to download new help files.</span></span>

1. <span data-ttu-id="22b4c-110">`Update-Help`sélectionne le fichier CAB pour le module à partir de l’emplacement spécifié par l’élément **HelpContentUri** dans le fichier XML HelpInfo distant.</span><span class="sxs-lookup"><span data-stu-id="22b4c-110">`Update-Help` selects the CAB file for the module from the location specified by the **HelpContentUri** element in the remote HelpInfo XML file.</span></span> <span data-ttu-id="22b4c-111">Elle utilise le nom du module, le GUID du module et la culture de l’interface utilisateur pour identifier le fichier CAB.</span><span class="sxs-lookup"><span data-stu-id="22b4c-111">It uses the module name, module GUID, and UI culture to identify the CAB file.</span></span>
