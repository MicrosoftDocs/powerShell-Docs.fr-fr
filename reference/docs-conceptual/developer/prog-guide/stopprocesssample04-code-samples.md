# <a name="stopprocesssample04-code-samples"></a>Exemples de code StopProcessSample04

Voici les exemples de code pour l’applet de commande StopProc00 Sample. Il s’agit de l’exemple d’applet de commande `Stop-Process` décrit dans [Ajout de jeux de paramètres à une applet](../cmdlet/adding-parameter-sets-to-a-cmdlet.md)de commande. L’applet de commande `Stop-Process` est conçue pour arrêter les processus qui sont récupérés à l’aide de l’applet de commande « obtient-proc » (décrit dans [création de votre première applet](../cmdlet/creating-a-cmdlet-without-parameters.md)de commande).

> [!NOTE]
> Vous pouvez télécharger le C# (stopprocesssample04.cs) et VB.net (stopprocesssample04. vb) pour cette applet de commande Stop-proc à l’aide du kit de développement logiciel (SDK) Microsoft Windows pour les composants d’exécution de Windows Vista et .NET Framework 3,0. Pour obtenir des instructions de téléchargement, consultez [Comment installer Windows PowerShell et télécharger le kit de développement logiciel (SDK) Windows PowerShell](/powershell/scripting/developer/installing-the-windows-powershell-sdk).
>
