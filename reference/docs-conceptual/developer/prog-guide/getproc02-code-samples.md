# <a name="getproc02-code-samples"></a>Exemples de code GetProc02

Voici les exemples de code pour l’applet de commande GetProc02 Sample. Il s’agit de l’exemple d’applet de commande `Get-Process` décrit dans [Ajout de paramètres qui traitent l’entrée de ligne de commande](../cmdlet/adding-parameters-that-process-command-line-input.md). Cette applet de commande `Get-Process` récupère les processus en fonction de leur nom, puis affiche des informations sur les processus sur la ligne de commande.

> [!NOTE]
> Vous pouvez télécharger le C# fichier source (getproc02.cs) pour cette applet de commande « obtenir-proc » à l’aide du kit de développement logiciel (SDK) Microsoft Windows pour les composants d’exécution de Windows Vista et .NET Framework 3,0. Pour obtenir des instructions de téléchargement, consultez [Comment installer Windows PowerShell et télécharger le kit de développement logiciel (SDK) Windows PowerShell](/powershell/developer/installing-the-windows-powershell-sdk).
>
> Les fichiers sources téléchargés sont disponibles dans le répertoire des **exemples \<PowerShell >** .
