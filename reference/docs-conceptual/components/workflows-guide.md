# <a name="psworkflow-module"></a>Module PSWorkflow

Cette section contient les rubriques d’aide relatives au module **PSWorkflow**. Celui-ci comprend des applets de commande qui prennent en charge la fonctionnalité Windows PowerShell Workflow. Le module **PSWorkflow** est installé avec Windows PowerShell 5.1.

> [!NOTE]
> Les workflows sont uniquement pris en charge dans Windows PowerShell 5.1.
