# <a name="contacting-package-owners"></a>Contacter les propriétaires de packages

Pour contacter le propriétaire d’un package particulier, accédez à la page de détail du package. Il existe un lien de contact des propriétaires dans la barre de menus de gauche. Cliquez sur ce lien pour accéder à une nouvelle page. Vous pouvez envoyer un message à partir de cette page.
