# Register-PackageSource

## SYNOPSIS
Ajoute une source de package pour un fournisseur de package spécifié.

## SYNTAX

### SourceBySearch

```
Register-PackageSource [-Proxy <Uri>] [-ProxyCredential <PSCredential>] [[-Name] <String>]
 [[-Location] <String>] [-Credential <PSCredential>] [-Trusted] [-Force] [-ForceBootstrap] [-WhatIf]
 [-Confirm] [-ProviderName <String>] [<CommonParameters>]
```

### NuGet

```
Register-PackageSource [-Proxy <Uri>] [-ProxyCredential <PSCredential>] [[-Name] <String>]
 [[-Location] <String>] [-Credential <PSCredential>] [-Trusted] [-Force] [-ForceBootstrap] [-WhatIf]
 [-Confirm] [-ConfigFile <String>] [-SkipValidate] [<CommonParameters>]
```

### PowerShellGet

```
Register-PackageSource [-Proxy <Uri>] [-ProxyCredential <PSCredential>] [[-Name] <String>]
 [[-Location] <String>] [-Credential <PSCredential>] [-Trusted] [-Force] [-ForceBootstrap] [-WhatIf]
 [-Confirm] [-PackageManagementProvider <String>] [-PublishLocation <String>]
 [-ScriptSourceLocation <String>] [-ScriptPublishLocation <String>] [<CommonParameters>]
```

## Description

L' `Register-PackageSource` applet de commande ajoute une source de package pour un fournisseur de package spécifié. Les sources de package sont toujours gérées par un fournisseur de package. Si le fournisseur de package ne peut pas ajouter ou remplacer une source de package, le fournisseur génère un message d’erreur.

## EXEMPLES

### Exemple 1 : inscrire une source de package pour le fournisseur NuGet

Cette commande inscrit une source de package, un emplacement basé sur le Web pour le fournisseur **NuGet** . Par défaut, les sources ne sont pas approuvées. Vous êtes invité à confirmer que la source est approuvée avant l’installation des packages. Pour remplacer la valeur par défaut, utilisez le `-Trusted` paramètre.

```powershell
Register-PackageSource -Name MyNuGet -Location https://www.nuget.org/api/v2 -ProviderName NuGet
```

```Output
Name          ProviderName     IsTrusted  Location
----          ------------     ---------  --------
MyNuGet       NuGet            False      https://www.nuget.org/api/v2
```
